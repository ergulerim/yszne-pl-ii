# yszne-pl-ii

Food Ordering System  
Software Development Methodologies Project

## Description
This project is a Food Ordering System developed as part of the Software Development Methodologies course.
The system demonstrates the use of RESTful APIs, layered architecture, and unit testing principles.

## Features
- List available food items
- REST API built with FastAPI
- Layered architecture (API, services, models)
- Unit testing using pytest

## Technologies Used
- Python
- FastAPI
- Pytest
- Uvicorn
